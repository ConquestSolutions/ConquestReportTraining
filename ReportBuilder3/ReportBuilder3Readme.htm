Report Builder 3.0 Readme

01/11/2010 14:02:00

This Readme document contains important information to read before installing Microsoft SQL Server 2008 R2 Report Builder 3.0. It describes how to find installation assistance, how to find supplemental product information in the Release Notes, and copyright attributions for the product.

This Readme file is also available from the Report Builder 3.0 download page.

What's New

For more information about the new features in Report Builder 3.0, see What's New in Report Builder 3.0.

Getting Assistance

The Report Builder Help file is the primary documentation for getting assistance with Report Builder. After you've installed Report Builder, you can view the Help content by clicking the blue question mark in the upper-left corner.

Before you install Report Builder, you can view the Report Builder Help content online. For more information installing Report Builder, see Installing Report Builder 3.0.

Read the Release Notes for the Latest Product Information

The Release Notes provide supplemental information about product behaviors that apply only to this release or that are not documented in this release of Report Builder Help. The Release Notes document is available from the Reporting Services section of the Microsoft SQL Server 2008 R2 Release Notes download page.
